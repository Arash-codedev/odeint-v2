Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/headmyshoulder/odeint-v2

odeint v1

The first version is also available as zip or tar.

Of course, you can clone odeint v1 with Git via: 

$ git clone -b odeint-v1 git://github.com/headmyshoulder/odeint-v2
